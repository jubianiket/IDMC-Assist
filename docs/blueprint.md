# **App Name**: IDMC Assist

## Core Features:

- AI-Powered Q&A: Generative AI answers Informatica IDMC questions based on user prompts, providing targeted assistance. The AI will have access to an internal knowledge tool.
- Prompt Input: A text input area for the user to input specific IDMC-related questions.
- Response Display: Displays the AI-generated answer in a clear, readable format.
- Feedback Collection: Ability for users to provide feedback on the quality and relevance of the AI's responses (e.g., thumbs up/down).
- AI Disclaimer: Include a disclaimer clarifying the AI's role as a learning assistant and that outputs may contain inaccuracies, requiring verification with official documentation.

## Style Guidelines:

- Primary color: A calm blue (#4681C4) to convey trustworthiness and competence.
- Background color: Light gray (#F0F4F8) to provide a clean and neutral backdrop.
- Accent color: A vibrant green (#62B462) to highlight important information or CTAs.
- Body and headline font: 'Inter', a grotesque-style sans-serif font, for a modern, machined, objective, neutral look; this font is appropriate for both headlines and body text.
